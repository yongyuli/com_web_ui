HTML5测试题整理①

发布时间：2020-06-05

1.在 HTML5 中，哪个元素用于组合标题元素？

答案：hgroup

2.HTML5 中不再支持哪个元素？

答案：font，acronym,applet,basefont,big,center,dir,frame,frameset,noframes,strike,tt

3.在 HTML5 中，onblur 和 onfocus 是?

答案：事件属性

4.在 HTML5 中，新多媒体元素都有哪些？代表什么含义？

audio：定义音频内容

video：定义视频（video 或者 movie）

source：定义多媒体资源 video 和 audio

embed：定义嵌入的内容，比如插件。

track：为诸如 video 和 audio 元素之类的媒介规定外部文本轨道。

5.在 HTML5 中不再支持？

答案：rel

6.在 HTML5 中，哪个方法用于获得用户的当前位置？

答案：getCurrentPosition()

7.新的 HTML5 全局属性，”contenteditable” 用于？

答案： 规定元素内容是否是可编辑的。

8.在 HTML5 中，contextmenu 和 spellcheck 是？

答案：HTML 属性

9.在 HTML5 中，您能够直接将 SVG 元素嵌入 HTML 页面中？

答案：正确。

10.由 SVG 定义的图形是什么格式的？

答案：XML格式。

11.HTML5 中的 canvas 元素用于？

答案：绘制图形。

12.哪个 HTML5 内建对象用于在画布上绘制？

答案：getContext。

13.在 HTML5 中，哪个属性用于规定输入字段是必填的？

答案：required。

14.哪种输入类型定义滑块控件？

答案：range。

15.哪种输入类型用于定义周和年控件（无时区）？

答案：week。

16.哪个 HTML5 元素用于显示已知范围内的标量测量？

答案：meter。

17.哪个标签用于指定内联框架？

答案： iframe

iframe 元素会创建包含另外一个文档的内联框架（即行内框架）。

18.SVG 有多少种图形对象类型？

答案：3。

在 SVG 中，允许三种图像对象存在，分别是矢量图像、点阵图像和文本。

19.HTML 5 支持 HTML 4 中的所有表单控件？

答案：正确。

20.DHTML是 ？

答案：DHTML 是一种使 HTML 页面具有动态特性的艺术，对大多数人来说，DHTML 意味着 HTML、CSS 和 JavaScript 的组合。
